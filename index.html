Array.prototype.forEach()


The forEach() method executes a provided function once for each array element. 


                
                    var numbers = [1,2,3];//input data

                    var sum = 0;
                    numbers.forEach(function (val) {
                        sum += val;
                    });

Array.prototype.reduce()


The reduce() method applies a function against an accumulator and each element in the array (from left to right) to reduce it to a value. 


                
                    var numbers = [1,2,3];//input data

                    var sum = numbers.reduce(
                        function (accumulator, value) {
                            return accumulator + value;
                        }
                    , 0); //initial accumulator = 0

Currying There is a way to reduce functions of more than one argument to functions of one argument, a way called currying after Haskell B. Curry. 
                
                    var curriedAdd = curry(
                        function add(a, b, c) {
                            return a + b + c;
                        }
                    );
                    curriedAdd(1)(2)(3);//=> 6

                    var add1 = curriedAdd(1);
                    var add1And2 = add1(2);
                    add1And2(3); //=> 6
Why Curry? -> function that can be partially configured, it encourages the creation of functions.

Put data the last argument, the arguments LESS likely to change first for code reuse.

Compose Function
a( b (x) ) = compose(a, b)(x) 
                
                    var compose = (a, b) => (x) => a( b (x) );

                    var number = compose(Math.round, parseFloat);
                    number('72.5'); //=> 73

                    var multiple = (a, b) => a * b;
                    var double = curry(multiple)(2);
                    var doubleNumber = compose(double, number);
                    doubleNumber('72.5'); //=> 146
